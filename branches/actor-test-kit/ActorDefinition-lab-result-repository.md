## ActorDefinition: Lab Result Repository 

| | |
| :--- | :--- |
| *Official URL*:http://mohcc.gov.zw/fhir/lab/ActorDefinition/lab-result-repository | *Version*:0.1.0 |
| Draft as of 2026-06-12 | *Computable Name*:LabResultRepository |

 
System that accepts ZWLabReportBundle documents from Lab Result Providers, stores them intact, and exposes results to Lab Result Consumers via FHIR search (DiagnosticReport, Bundle). In the Zimbabwe national architecture this role is played by the Shared Health Record (HAPI FHIR), fronted by OpenHIM.
