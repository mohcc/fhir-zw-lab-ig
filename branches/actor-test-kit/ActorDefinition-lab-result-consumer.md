## ActorDefinition: Lab Result Consumer 

| | |
| :--- | :--- |
| *Official URL*:http://mohcc.gov.zw/fhir/lab/ActorDefinition/lab-result-consumer | *Version*:0.1.0 |
| Draft as of 2026-06-12 | *Computable Name*:LabResultConsumer |

 
System that queries the Lab Result Repository for results for its patients (FHIR search on DiagnosticReport / report Bundles — result pull) and presents them to clinical users. In the Zimbabwe national architecture this role is played by the Impilo EHR.
